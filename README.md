# Linktree

## By Maurice Nganga

## Description

---
This is a frontend of a a page like Linktree that allows a person show their personal profile and link to various digital things that they sell

The front-end is diployed in [vercel](https://vercel.com/) hosting service .

## Features

---


### Technology Used

---

- **Styled-components** - framework was used to add styling of the website.

- **React** - help in creating interactive website
